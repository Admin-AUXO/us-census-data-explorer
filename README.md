# US Census Data Explorer

Interactive visualization of US Census ACS data at multiple geographic levels.

**AUXO Data Labs** | [auxodata.com](https://auxodata.com)

## Features

- Multi-level geographic analysis (State → County → ZIP Code)
- Comprehensive statistical insights
- Interactive data visualization
- Year-over-year comparisons
- Advanced filtering capabilities

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Data Source

US Census Bureau - American Community Survey (ACS) 5-Year Estimates
